# Colegio De Muntinlupa Information System

**Note: This is an academic project developed as a requirement for the Object-Oriented Programming (OOP) subject. It is intended for educational and demonstration purposes only.**

A desktop-based informational portal designed to provide users with a comprehensive overview of Colegio De Muntinlupa (CDM). This project serves as a practical application of Java Swing for GUI development and OOP design patterns.

## 📌 Project Overview
The system provides a navigable interface where users can explore university details, including the mission/vision, available engineering courses (Civil, Industrial, Computer Engineering, etc.), and campus facilities.

### Key Features
* **Multi-Page Navigation**: A centralized main menu allowing users to toggle between Home, About, Courses, and Facilities pages.
* **Interactive GUI**: Built using Java Swing components (`JFrame`, `JPanel`, `JButton`) for a user-friendly desktop experience.
* **OOP Implementation**: Utilizes class-based architecture to separate the main logic, frame setups, and page transitions.

## 🛠️ Technical Stack
* **Language**: Java
* **Framework**: Java Swing / AWT
* **IDE**: NetBeans

## 🏗️ OOP Concepts Applied
This project demonstrates several core concepts learned in the Object-Oriented Programming subject:
* **Encapsulation**: State and behavior are bundled within specific page and frame classes.
* **Abstraction**: Hiding complex GUI rendering logic behind simple method calls like `MainFrameClassSetUp()`.
* **Modularity**: Separation of concerns by splitting the Main Class, Main Frame, and individual page panels.
